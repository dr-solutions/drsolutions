IonicPage Beispiel Die Page hat ein eigenes Modul. Dieses wird in App Modules injected (Das Modul und nicht die Page ansich). Die Page selbst hat @IonicPage decorator Properties für diesen Decorator: name: 'meinePage' segment: 'myPage' // Wenn segement nicht angegeben wird wird name verwendet Zu segement Propertie können auch 'Url-Params' angegeben werden' z.b.: segment: 'myPage:id' Diese muss dann beim wechsel (this.navCtrl.push) angegeben werden defaultHistory: ['list'] defaultHistory wird verwendet um mit den Back Button speziell hin zu navigieren. z.B.: Back button würde jetzt zu list wechseln priority: 'high' Default priority: 'off'. High bedeutet beim App start die Page schon laden (nicht erst beim Klick auf die Page).
